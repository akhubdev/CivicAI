# 🌍 Overview

CivicAI is an AI-powered web platform developed to modernize the way citizens report public infrastructure issues.

Instead of manually writing long complaints, users simply upload an image of a civic problem. Google Gemini AI automatically analyzes the image and generates a complete report including issue type, severity, department, priority level, confidence score and recommended action.

The project aims to bridge the communication gap between citizens and government authorities by making civic issue reporting intelligent, fast and data-driven.

---

# 🎯 Problem Statement

Traditional complaint systems suffer from several limitations:

- Manual reporting process
- Slow response time
- Incorrect department selection
- No priority classification
- Lack of visual evidence analysis
- Poor analytics for authorities

CivicAI solves these problems using Artificial Intelligence and Computer Vision.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/akhubdev/CivicAI.git
```

---

## Go to Project Folder

```bash
cd CivicAI
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

---

### Linux / macOS

```bash
python3 -m venv venv
```

Activate

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Create .env File

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

---

## Run Project

```bash
python app.py
```

---

Open your browser

```
http://127.0.0.1:5000
```
